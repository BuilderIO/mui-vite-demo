# Hello World Page

A welcoming introduction page built with Material-UI, featuring modern design patterns and interactive elements.

## Features

- **Animated Welcome Icon**: A waving hand icon with smooth rotation animation
- **Gradient Typography**: Eye-catching gradient text effect for the main heading
- **Interactive Button**: Celebration mode with icon and text changes
- **Technology Chips**: Visual indicators showing the tech stack
- **Responsive Design**: Adapts to different screen sizes
- **Theme Support**: Full support for light and dark modes
- **Color Mode Toggle**: Top-right corner theme switcher

## Components Used

- Material-UI Card and CardContent for layout
- Typography with custom styling and gradients
- Stack for organized spacing
- Chip components for technology badges
- Button with start icons and hover effects
- Custom styled components with theme integration

## Routing

The page is accessible at:
- `/` (homepage)
- `/hello` (direct route)

## Styling Features

- Custom background gradient overlay
- Theme-aware colors that adapt to light/dark mode
- Smooth animations and transitions
- Responsive typography scaling
- Material Design elevation and shadows

## Customization

The component accepts the same props as other pages in this project:
- `disableCustomTheme?: boolean` - Option to disable custom theming

The design follows the established patterns from other components in the project while showcasing Material-UI's capabilities.
